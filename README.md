# K-Means Clustering Data Science Project

## 📌 Project Overview
This project demonstrates the implementation of the **K-Means Clustering Algorithm** using Python and Machine Learning libraries.  
The project covers the complete workflow of clustering including:

- Data preprocessing
- Feature scaling
- Finding the optimal value of K
- Training the K-Means model
- Cluster visualization
- Model evaluation

This project helps in understanding how **Unsupervised Machine Learning** works for grouping similar data points into clusters.



## 🚀 Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
